Storage Design

Blob Storage Tiers
HotFrequent access — highest storage cost, lowest access cost
CoolInfrequent (30d+) — lower storage, higher access
ColdRare access (90d+) — even lower storage cost
ArchiveOffline (180d+) — rehydrate hours before access
Blob Features
Lifecycle policiesAuto-tier or delete based on age/last access
Immutable storageWORM: time-based retention + legal hold
VersioningKeep previous versions on overwrite/delete
ReplicationObject replication to another account/region
Storage Redundancy
LRS3 copies, single datacenter — 99.999999999%
ZRS3 copies, 3 zones — recommended for HA
GRSLRS + async copy to paired region
GZRSZRS + async copy to paired region — highest
Storage Decision Tree
Unstructured data/blobsAzure Blob Storage
File shares (SMB/NFS)Azure Files + Azure File Sync
VM disksAzure Managed Disks (Premium SSD/Ultra)
Big data/analyticsAzure Data Lake Storage Gen2
High-perf NFSAzure NetApp Files
Azure File Sync: tier cold files to cloud, keep hot files on-prem. ZRS is the default recommendation for new storage accounts in regions that support it.

BCDR Fundamentals
RTORecovery Time Objective — max downtime acceptable
RPORecovery Point Objective — max data loss acceptable

Active-PassiveSecondary warm/cold standby — lower cost
Active-ActiveBoth regions serve traffic — zero downtime

VM backupDaily/weekly/monthly/yearly retention policies

RPO for Azure VMs30 seconds (continuous replication)
FailoverTest failover (no impact) → planned → unplanned
FailbackReplicate back and fail back to primary
RunbooksAutomation runbooks for orchestrated failover
On-prem to AzureHyper-V, VMware, physical server replication
Traffic Routing — Decision
Traffic ManagerDNS-based global LB — priority/weighted/performance/geo
Azure Front DoorLayer 7 global LB + WAF + caching + SSL offload
App GatewayRegional Layer 7 LB + WAF — within VNet
Azure Load BalancerLayer 4 regional — VMs, VMSS within region
Traffic Manager is DNS-level only — no connection failover. Front Door = global L7. Use both together: Front Door for web apps, Traffic Manager for non-HTTP globally distributed services.
