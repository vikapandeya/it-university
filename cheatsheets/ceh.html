M03 — Scanning Networks

Module 3
Nmap Scan Types
-sSSYN (stealth) scan — default
-sTTCP Connect scan — full 3-way
-sUUDP scan — slow, stealthy
-sNNULL scan — no flags set
-sFFIN scan — FIN flag only
-sXXmas scan — FIN+PSH+URG
-sAACK scan — firewall mapping
-sVVersion detection
-OOS fingerprinting
-p-All 65535 ports
-AAggressive: OS+version+scripts
Banner Grabbing
nc -vn 192.168.1.1 80 telnet 192.168.1.1 25
Hping3 Examples
hping3 -S -p 80 target.com hping3 --udp -p 53 target.com
NULL, FIN, Xmas scans: Open port = no response; Closed port = RST. Works only on Unix/Linux targets — Windows ignores these scans.

Module 20

3DES168-bit; DES × 3; deprecated
AES128/192/256-bit; current standard
Blowfish64-bit block; 32–448-bit key
RC4Stream cipher; used in WEP/SSL (broken)
TwofishAES finalist; 128-bit block
Asymmetric Encryption
RSA2048+ bit; encrypt & sign
ECCElliptic curve; smaller keys, fast
DHDiffie-Hellman key exchange
DSADigital Signature Algorithm
ElGamalEncryption + signatures
Hashing Algorithms
MD5128-bit; broken; collision-prone
SHA-1160-bit; deprecated; broken
SHA-256256-bit; current standard
SHA-3Keccak; latest NIST standard
HMACHash + secret key; integrity+auth
bcryptSlow hash; adaptive work factor
PKI & SSL/TLS
CACertificate Authority; issues certs
RARegistration Authority; verifies identity
CRLCertificate Revocation List
OCSPOnline Certificate Status Protocol
TLS 1.3Current; removed weak ciphers
Steganography vs Cryptography
CryptographyHides message content (unreadable)
SteganographyHides message existence (invisible)
Digital Signature: Hash the message → encrypt hash with sender's private key → recipient decrypts with sender's public key to verify integrity & authenticity.
